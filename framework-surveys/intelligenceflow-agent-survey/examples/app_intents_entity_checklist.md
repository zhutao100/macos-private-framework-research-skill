# App Intents entity/action checklist

For each App Intent trial, record:

- App name and bundle identifier.
- Intent type name.
- Assistant Schema/domain if adopted.
- Parameters and entity properties.
- Whether output is text, dictionary, rich text, entity, or app-specific content.
- Whether the action is visible in Shortcuts.
- Whether the action is visible in Spotlight.
- Whether Siri can invoke it.
- Whether Shortcuts Use Model can consume its output.
- Trial prompt/action and exact timing.
- Logs and privacy-report side effects.
